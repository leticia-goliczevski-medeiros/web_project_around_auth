## Descrição
É uma página interativa, onde o usuário pode adicionar, remover ou curtir fotos. Ele também pode criar um perfil com foto e informações básicas.

## Tecnologias
A parte de estilos foi escrita em CSS puro. O projeto foi criado com o Vite e a biblioteca React foi utilizada na renderização dos componentes e controle de estado. A biblioteca React Hook Form foi utilizada na validação dos formulários.
